# medicare-hospital-management
Hospital Management System
Java, Maven, Spring Core, JDBC CRUD. Role-based login for Admin, Doctor, and Patient.

## Modules
- Auth (Login/Register, Roles)
- Patient (Profile, Appointments, Reminders, Pharmacies)
- Doctor (Profile, Appointments)
- Admin (Manage Users, Doctors, Patients)

## Tech
Java 17, Maven, Spring Core, Spring JDBC, MySQL
